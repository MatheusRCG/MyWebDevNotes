Para acrescentar estilo a nossa pagina usamos o CSS, basicamente temos 3 formas de usa-lo:

Tipos de CSS

O CSS pode ser usado de 3 formas, inline, interno e externo, cada uma dessas formas tem suas vantagens e desvantagens.

CSS Inline

Podemos adicionar um estilo na própria criação do elemento, pra isso usamos o parâmetro style="codigo css"

Código:

          
            <p style="background-color: red; color: white;">texto do paragrafo</p>

Resultado:

texto do paragrafo

  • Para modificarmos a cor do fundo usamos: background-color: cor_desejada
  • E para modificarmos a cor da letra usamos: color: cor_desejada
  • Quando temos mais de uma modificação, separamos elas por ;
  • Com css podemos modificar muito mais do que apenas cores...

CSS Interno

Usar css direto no elemento nem sempre é interessante, uma vez que se tivermos diversos elementos que queremos que o mesmo estilo seja aplicado, teriamos que repetir o mesmo style na criação de todos, uma forma de contornar isso é usando o css interno, pra que isso seja feito, usamos a tag <style>codigo css>/style> dentro da tag head de nossa página.

Código:

          
            <head>
              <style>
                p{
                  background-color: blue;
                  color: white;
                }
              </style>
            </head>
            <body>
              <p>texto do paragrafo 1</p>
              <p>texto do paragrafo 2</p>
              <p>texto do paragrafo 3</p>
            </body>

Resultado:

texto do paragrafo 2

texto do paragrafo 1

texto do paragrafo 3

Dessa forma, todos os paragrafos dessa página teriam o mesmo estilo. Isso também poderia ser feito com outras tags html como <h1> ou <div> por exemplo.

CSS Externo

Para usarmos o css externo precisamos criar um arquivo .css que terá todo o codigo de estilo, depois precisamos usar a tag <link rel="stylesheet" href="./endereco_do_arquivo.css"> dentro do <head>

Código:

Arquivo style.css
          
            p{
              background-color: yellow;
            }
Arquivo pagina_1.html
          
            <html>
              <head>
                <link rel="stylesheet" href="./style.css">
              </head>
              <body>
                <p>texto do paragrafo da página 1</p>
              </body>
            </html>
Arquivo pagina_2.html
          <html>
              <head>
                <link rel="stylesheet" href="./style.css">
              </head>
              <body>
                <p>texto do paragrafo da página 2</p>
              </body>
            </html>

Resultado:

texto do paragrafo da página 1

texto do paragrafo da página 2

Dessa forma, podemos usar o mesmo arquivo em quantas paginas html quisermos passando apenas o link do arquivo css dentro da tag head em cada uma das páginas.

Elemento, Classes e Id's

Além de modificar o estilo de tags html, podemos criar classes e id's para que possamos modificar o estilo de um grupo de elementos ou de um elemento específico.

Código

Arquivo index.html

        
          <html>
            <head>
              <link rel="stylesheet" href="./style.css">
            </head>
            <body>
              <p>texto do paragrafo 1</p>
              <p class="paragrafo">texto do paragrafo 2</p>
              <p id="paragrafo_3">texto do paragrafo 3</p>
            </body>
          </html>

Arquivo style.css

        
          p{
            background-color: yellow;
          }
          .paragrafo{
            background-color: green;
          }
          #paragrafo_3{
            background-color: blue;
            color: white;
          }

Resultado

texto do paragrafo 1

texto do paragrafo 2

texto do paragrafo 3

Cores

Para modificar a cor de fundo de um elemento usamos a propriedade background-color: cor_desejada e para modificar a cor do texto usamos a propriedade color: cor_desejada

Cores Padrões

a primeira forma de modificar a cor é usando uma das cores padrões (red, blue, yellow, green, etc...) Ao todo são 140 cores prontas para serem usadas. 

          <p style="background-color: yellow;">paragrafo amarelo</p>.

Resultado

paragrafo amarelo

Cores usando RGB

Outra forma de modificar a cor é usando o RGB, que é uma forma de representar a cor através de 3 valores, o primeiro é a quantidade de vermelho, o segundo é a quantidade de verde e o terceiro é a quantidade de azul, cada um desses valores varia de 0 a 255. 

          <p style="background-color: rgb(255, 0, 0);">paragrafo vermelho</p>

Resultado

paragrafo vermelho

Opacidade

Podemos também modificar a opacidade da cor, para isso usamos o RGBA, que é a mesma coisa que o RGB, mas com um quarto valor que representa a opacidade, esse valor varia de 0 a 1. 

          
            <p style="background-color: rgba(0, 128, 0, 0.8);">paragrafo verde com opacidade</p>

Resultado

paragrafo verde com opacidade

Cores usando HEXdecimal

A ultima forma de modificar a cor é usando o HEX, que é uma forma de representar a cor através de 6 valores, os 2 primeiros representam a quantidade de vermelho, os 2 do meio representam a quantidade de verde e os 2 últimos representam a quantidade de azul, cada um desses valores varia de 00 a FF. 

          <p style="background-color: #00FF00;">paragrafo verde</p>

Resultado

paragrafo verde

Background Cover

usaremos agora esta imagem como fundo para o mesmo tamanho de div, poderiamos também usar o background-repeat: no-repeat para a imagem não se repetir mas caso o fundo seja maior que a imagem, ela não preencherá o espaço todo, utilizaremos então o parametro background-size: cover para que a imagem preencha toda a div

A imagem se ajusta ao tamanho da div porém assim que a imagem alcançar a proporção da div ela começará a se repetir.

Background Gradient

Além de imagens, podemos usar gradientes como fundo de um elemento, formando um degrade com as cores, para isso usamos a propriedade background no css passando a função linear-gradient como valor. background: linear-gradient(direção, cor1, cor2)

Exemplos

Usando background: linear-gradient(to right, red, yellow) temos um degradê da equerda para a direita do vermelho ao amarelo

Usando background: linear-gradient(to right, red, 20%, yellow) podemos dizer onde iniciará a cor amarela, no exemplo ela começa em 20% da largura do elemento.

Usando background: linear-gradient(to right, red, yellow 20%) podemos dizer onde ficará a parte mais forte da cor amarela, no exemplo ela começa/fica mais forte em 20% da largura do elemento.

Usando background: linear-gradient(to left bottom, red, yellow) posso modificar a direção do degradê, nesse caso ele vai da direita para a esquerda e de cima para baixo.

Usando background: linear-gradient(to right, red, rgb(0,200,0), #FFFF00) posso selecionar mais de uma cor para o degradê. Alem disso posso passar rgb ou hex como cor.

por fim se não for passado direção para o degradê, background: linear-gradient(red, yellow) ele será feito de cima para baixo. sendo equivalente a to bottom como direção

Unidades de medidas

Além de cores e imagens, podemos modificar o tamanho de um elemento, para isso usamos a propriedade width e height no css, podemos passar valores em px, em, rem, %, vh e vw.

Pixel

O px é a unidade de medida mais comum, ela é fixa e não muda de acordo com o tamanho da tela. porem 2 telas podem ter o mesmo tamanho (cemtimetros) mas resoluções diferentes, fazendo com que o mesmo elemento tenha tamanhos diferentes. ou seja, 50px em uma tela de 800px de largura é diferente de 50px em uma tela de 1600px de largura quando visto a olho nu. 

          
            <div style="width: 100px; height: 100px"></div>

Porcentagem

A porcentagem é uma unidade de medida relativa, ou seja, ela é calculada com base em outra unidade de medida, por exemplo, se um elemento tem 50% de largura, ele terá metade da largura do elemento pai. 

          
            #divVermelha { 
              width: 200px; 
              height: 100px"
            }
            #divAzul { 
              width: 100px; 
              height: 100px"
            }
            #divAmarela {
              width: 50%;
              height: 50%"
            }

Perceba que a div amarela ocupa 50% do tamanho da div PAI tanto horizontalmente quanto verticalmente.

Viewport

A unidade vh e vw são relativas ao tamanho da tela, 1vh é 1% da altura da tela e 1vw é 1% da largura da tela. 

            
              #divVermelha { 
                width: 20vw; 
                height: 10vh"
              }
              #divAzul { 
                width: 10vw; 
                height: 10vh"
              }
              #divAmarela {
                width: 50%;
                height: 50%"
              }

Esse exemplo só pode ser observado a medida que a tela é redimensionada, pois as div vermelha e azul tem tamanhos relativos a tela(navegador). assim que o navegador e redimencionado as div vermelha e azul mudam de tamanho por estarem relacionadas ao tamanho da tela e consequentemente a div amarela também muda de tamanho por estar usando a porcentagem relacionada a div pai.

Unidade de medida para Texto

Podemos utilizar o pixel para definir o tamanho da fonte, mas também podemos usar o em e o rem, o em é relativo ao tamanho da fonte do elemento pai, já o rem é relativo ao tamanho da fonte do elemento raiz. 

          
            #divVermelha { 
              font-size: 20px;
            }
            #divAzul { 
              font-size: 2em;
            }
            #divAmarela {
              font-size: 2rem;
            }
O texto dessa div está com 20px
O texto dessa div está com 2em = 40px
O texto dessa div está com 2rem = 32px

Perceba que a div azul tem o dobro do tamanho da fonte da div vermelha, pois o em é relativo ao tamanho da fonte do elemento pai, já a div amarela tem o dobro do tamanho da fonte da div vermelha, pois o rem é relativo ao tamanho da fonte do elemento raiz que por padrão é 16px.

Margin e Padding

Margin

A propriedade margin é usada para criar espaço ao redor dos elementos, fora das bordas. Podemos definir margens para os quatro lados de um elemento: superior, inferior, esquerdo e direito.

Usando margin: valor podemos definir a margem de todos os lados de uma vez. Por exemplo, margin: 10px define uma margem de 10px em todos os lados.

Elemento 1

Estes elemento tem uma margem de 10px em todos os lados.

elemento 3

Podemos também definir margens específicas para cada lado usando as propriedades margin-top, margin-right, margin-bottom e margin-left ou apenas apenas passando cada lado na propriedade margin. começando do topo e seguindo no sentido horário. margin: 10px 20px 30px 40px

Padding

A propriedade padding é usada para criar espaço ao redor do conteúdo dentro de um elemento, dentro das bordas. Assim como a margem, podemos definir preenchimentos para os quatro lados de um elemento: superior, inferior, esquerdo e direito.

Usando padding: valor podemos definir o preenchimento de todos os lados de uma vez. Por exemplo, padding: 10px define um preenchimento de 10px em todos os lados.

Este elemento tem um preenchimento de 20px em todos os lados.

Assim como margin podemos também definir padding específicos para cada lado usando as propriedades padding-top, padding-right, padding-bottom e padding-left ou apenas apenas passando cada lado na propriedade padding. começando do topo e seguindo no sentido horário. padding: 10px 20px 30px 40px

box sizing

A propriedade box-sizing é usada para alterar o modelo de caixa padrão do CSS. O modelo de caixa padrão é o modelo de caixa de conteúdo, onde a largura e a altura de um elemento definem o tamanho do conteúdo da caixa. A propriedade box-sizing pode ter os seguintes valores:

  • content-box - O tamanho da caixa é a largura e a altura do conteúdo. Isso é o padrão.
  • border-box - O tamanho da caixa é a largura e a altura do conteúdo + padding + borda.

Usando box-sizing: border-box podemos fazer com que a largura e a altura de um elemento incluam o preenchimento e a borda, em vez de apenas o conteúdo.

Este elemento tem uma largura de 300px, um preenchimento de 20px e uma borda de 2px. como está utilizando o border-box então totaliza o conteúdo é ajustado para que o total seja 300px de largura e 300px de altura.

Overflow

A propriedade overflow é usada para controlar o que acontece se o conteúdo de um elemento for maior do que a caixa que o contém. Ela pode ter os seguintes valores:

  • visible - O conteúdo é exibido fora da caixa.
  • hidden - O conteúdo é cortado e não é exibido fora da caixa.
  • scroll - O conteúdo é cortado, mas um scroll é adicionado para ver o resto do conteúdo.
  • auto - Caso o conteúdo for maior que a caixa, um scroll é adicionado para ver o resto do conteúdo.

Usando overflow: hidden podemos fazer com que o conteúdo que ultrapassa a caixa seja cortada e não seja exibida fora da caixa.

Usando overflow: auto podemos fazer com que o conteúdo que ultrapassa a caixa seja cortado e um scroll seja adicionado para ver o resto do conteúdo.

Este elemento tem uma largura de 200px e uma altura de 100px. O conteúdo que ultrapassa a caixa é cortado, mas usando o overflow: auto, um scroll é adicionado para ver o resto do conteúdo e caso o conteúdo não ultrapasse a caixa, o scroll não é exibido.

Listas

Listas são usadas para agrupar itens relacionados. Existem dois tipos principais de listas em HTML: listas ordenadas e listas não ordenadas.

Listas Não Ordenadas

Listas não ordenadas são usadas para agrupar itens sem uma ordem específica. Elas são criadas com a tag <ul> e cada item da lista é criado com a tag <li>. 

          
            <ul>
              <li>Item 1</li>
              <li>Item 2</li>
              <li>Item 3</li>
            </ul>

Resultado:

  • Item 1
  • Item 2
  • Item 3

Listas Ordenadas

Listas ordenadas são usadas para agrupar itens em uma ordem específica. Elas são criadas com a tag <ol> e cada item da lista é criado com a tag <li>. 

          
            <ol>
              <li>Item 1</li>
              <li>Item 2</li>
              <li>Item 3</li>
            </ol>

Resultado:

  1. Item 1
  2. Item 2
  3. Item 3

Estilizando Listas com Imagens

Podemos usar imagens como marcadores de lista usando a propriedade list-style-image e passando o caminho da imagem. O problema é que essas imagens não são responsivas, então é nescessário usar uma imagem do tamnho da fonte, no caso estamos usando uma fonte de 16px então precisamos de uma imagem de 16x16px. 

          
            ul {
              list-style-image: url('caminho/para/imagem.png');
            }

Lista estilizada

Podemos estilizar as listas de várias maneiras, como alterar a cor, o tamanho e o estilo da fonte, adicionar margens e preenchimentos, adicionar bordas e muito mais. 

          
            ul {
              list-style: none;
              padding: 5px;
              max-width: 400px;
              box-shadow: 4px 4px 10px #717e88;
              border-radius: 15px;
            }
            
            ul li {
              padding: 10px 20px;
              margin: 5px;
              font-size: 18px;
              color: black;
              background-color: #4fb7fc;
              border: 1px solid #007ff7;
              border-radius: 25px;
            }
            
            ul li:hover {
              color: white;
              background-color: #007ff7;
              margin: 0px;
              cursor: pointer;
            }
