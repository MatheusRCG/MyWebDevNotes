Pseudo-Elementos

Pseudo-elementos são palavras-chave adicionadas a seletores que permitem que você estilize partes específicas de um elemento. Eles são usados para aplicar estilos a partes de um elemento que não podem ser acessadas diretamente pelo HTML.

::first-line

O pseudo-elemento ::first-line é usado para aplicar estilos à primeira linha de um bloco de texto. 

          
            p::first-line {
              font-weight: bold;
            }

Selectores

Os seletores são usados para selecionar elementos HTML com base em seus atributos, classes, IDs, tipos, etc.

Seletores Básicos

Os seletores básicos são usados para selecionar elementos HTML com base em seus nomes de tag, classes e IDs. 

          
            /* Seletor de tag */
            p {
              color: blue;
            }

            /* Seletor de classe */
            .example-class {
              color: red;
            }

            /* Seletor de ID */
            #example-id {
              color: green;
            }
  • p seleciona todos os elementos p.
  • .example-class seleciona todos os elementos com a classe example-class.
  • #example-id seleciona o elemento com o ID example-id.

Seletores Combinadores

Os seletores combinadores são usados para selecionar elementos com base na relação entre eles. 

          
            /* Seletor de filho direto */
            div > p {
              color: purple;
            }

            /* Seletor de todos os descendentes */
            div p {
              color: orange;
            }

            /* Seletor de irmão adjacente */
            h1 + p {
              color: pink;
            }

            /* Seletor de irmãos gerais */
            h1 ~ p {
              color: brown;
            }
  • div > p seleciona todos os p que são filhos diretos de um div.
  • div p Seleciona todos os elementos que são descendentes de um elemento específico, não importando o quão profundamente eles estão aninhados.
  • h1 + p seleciona o p que vem imediatamente após um h1.
  • h1 ~ p seleciona todos os p que são irmãos de um h1 e que aparecem depois dele.

Seletores de Atributo

Os seletores de atributo são usados para selecionar elementos com base em seus atributos e valores de atributos. 

          
            /* Seletor de atributo */
            [title] {
              color: teal;
            }

            /* Seletor de atributo com valor específico */
            [title="example"] {
              color: navy;
            }

            /* Seletor de atributo contendo valor */
            [title*="ex"] {
              color: olive;
            }
  • [title] seleciona todos os elementos que têm um atributo title.
  • [title="example"] seleciona todos os elementos que têm um atributo title com o valor example.
  • [title*="ex"] seleciona todos os elementos que têm um atributo title com um valor que contém a substring ex.

Pseudo-Classes

Pseudo-classes são palavras-chave adicionadas a seletores que especificam um estado especial do elemento a ser selecionado.

:first-child e :last-child

As pseudo-classes :first-child e :last-child são usadas para selecionar o primeiro e o último filho de um elemento pai, respectivamente. 

          
            /* Seleciona o primeiro filho */
            p:first-child {
              color: blue;
            }

            /* Seleciona o último filho */
            p:last-child {
              color: red;
            }

:nth-child

A pseudo-classe :nth-child é usada para selecionar um ou mais filhos com base em sua posição no elemento pai. Pode usar fórmulas como 2n, 3n, odd (ímpar) e even (par). 

          
            /* Seleciona todos os filhos em posições pares */
            p:nth-child(even) {
              background-color: green;
            }

            /* Seleciona todos os filhos em posições ímpares */
            p:nth-child(odd) {
              background-color: purple;
            }

            /* Seleciona especificamente o segundo elemento */
            p:nth-child(2) {
              color: white;
            }

            /* Seleciona cada terceiro filho */
            p:nth-child(3n) {
              color: orange;
              font-weight: bold;
            }

:focus

A pseudo-classe :focus é usada para selecionar um elemento que está em foco, como um campo de entrada de texto. 

          
            input:focus {
              color: white;
              background-color: blue;
            }

Diferença entre Pseudo-Classes e Pseudo-Elementos

Pseudo-classes são usadas para selecionar elementos com base em seu estado ou posição no DOM, enquanto pseudo-elementos são usados para estilizar partes específicas de um elemento.

  • Pseudo-Classes: :hover, :focus, :nth-child, etc.
  • Pseudo-Elementos: ::before, ::after, ::first-line, etc.

Função Calc

            div {
              width: calc(100% - 100px);

Variaveis

As variáveis CSS são usadas para armazenar valores que podem ser reutilizados em todo o documento.

São criadas usando a propriedade --nome-variavel dentro de :root e são acessadas usando a função var(). 

        
          :root {
            --cor-primaria: blue;
            --cor-secundaria: green;
          }

          div {
            background-color: var(--cor-primaria);
            color: var(--cor-secundaria);
          }

Essas variáveis são criadas dentro de :root porque :root é um pseudo-seletor que representa o elemento raiz do documento, geralmente o elemento <html>. Isso significa que as variáveis definidas dentro de :root estarão disponíveis globalmente em todo o documento, permitindo que sejam reutilizadas em qualquer parte do CSS.

FlexBox

O Flexbox é um modelo de layout unidimensional que permite criar layouts complexos e flexíveis sem usar floats ou layouts baseados em grade.

Para usar o Flexbox, você precisa definir um elemento pai como um container flex defindo o display como flex. Isso transforma todos os filhos diretos desse elemento em itens flexíveis.

Flex Direction

A propriedade flex-direction é usada para definir a direção principal dos itens flexíveis. Pode ser row (valor padrão), row-reverse, column ou column-reverse. 

          
            .container {
              display: flex;
              flex-direction: row;
            }

Flex Wrap

A propriedade flex-wrap é usada para definir se os itens flexíveis devem ser dispostos em uma única linha ou várias linhas.

Usando como valor wrap, os itens flexíveis são dispostos em uma única linha, mas se o espaço for insuficiente, eles serão quebrados em várias linhas. caso não seja passado nenhum valor para a propriedade, o valor padrão é nowrap. 

          
            .container {
              display: flex;
              flex-wrap: wrap;
            }

Justify-content

A propriedade justify-content é usada para alinhar os itens flexíveis ao longo do eixo principal do contêiner flexível. Os valores possíveis para justify-content são:

  • flex-start (valor padrão): Alinha os itens ao início do contêiner.
  • flex-end: Alinha os itens ao final do contêiner.
  • center: Alinha os itens ao centro do contêiner.
  • space-between: Distribui os itens com espaço igual entre eles.
  • space-around: Distribui os itens com espaço igual ao redor deles.
  • space-evenly: Distribui os itens com espaço igual entre eles e nas extremidades.
          
            .container {
              display: flex;
              justify-content: space-evenly;
            }

Align Itens

A propriedade align-items é usada para alinhar os itens flexíveis ao longo do eixo transversal do contêiner flexível. Os valores possíveis para align-items são:

  • stretch (valor padrão): Estica os itens para preencher o contêiner.
  • flex-start: Alinha os itens ao início do contêiner.
  • flex-end: Alinha os itens ao final do contêiner.
  • center: Alinha os itens ao centro do contêiner.
  • baseline: Alinha os itens à linha de base do contêiner.

Grow / Shrink

As propriedades flex-grow e flex-shrink são usadas para definir como os itens flexíveis devem crescer ou encolher para preencher o espaço disponível. Ambos aceitam valores numéricos, onde 0 significa que o item não deve crescer ou encolher e valores maiores significam que o item deve crescer ou encolher proporcionalmente mais.

A propriedade flex-grow define a capacidade de um item flexível de crescer em relação aos outros itens flexíveis no contêiner.

A propriedade flex-shrink define a capacidade de um item flexível de encolher em relação aos outros itens flexíveis no contêiner. 

          
            .item {
              flex-grow: 1;
              flex-shrink: 1;
            }

Grid

O CSS Grid Layout é um sistema de layout bidimensional que permite criar layouts complexos e flexíveis sem usar floats ou layouts baseados em grade.

Para usar o Grid, você precisa definir um elemento pai como um container grid definindo o display como grid. Isso transforma todos os filhos diretos desse elemento em itens de grade.

Grid Template Columns

A propriedade grid-template-columns é usada para definir o número e a largura das colunas em um contêiner de grade. 

          
            .container {
              display: grid;
              grid-template-columns: 20% 50% 30%;
              grid-gap: 1px;
            }

Grid Template

A propriedade grid-template é uma propriedade abreviada que combina grid-template-rows e grid-template-columns.

Também é possível definir o grid usando a unidades de medida fr, que é uma unidade de medida flexível que distribui o espaço disponível proporcionalmente. Por exemplo, 1fr 2fr 1fr distribui o espaço disponível em 4 partes, onde a primeira e a terceira partes têm o mesmo tamanho e a segunda parte tem o dobro do tamanho.

              display: grid;
              grid-template: 2fr 1fr / 1fr 2fr 1fr

Grid Colum e Row Start e End

As propriedades grid-column-start, grid-column-end, grid-row-start e grid-row-end são usadas para definir onde um item de grade deve começar e terminar em relação às linhas e colunas de grade. 

          
            .container {
              display: grid;
              gap: 1px;
              color: white;
              text-align: center;
            }
            
            .item-1 {
              grid-column-start: 1;
              grid-column-end: span 3;
            }
            
            .item-2 {
              grid-column-start: 2;
              grid-column-end: span 2;
            }
            
            .item-3 {
              grid-row-start: 2;
            }
            
            .item-4 {
              grid-row-start: 3;
              grid-column-end: span 3;
