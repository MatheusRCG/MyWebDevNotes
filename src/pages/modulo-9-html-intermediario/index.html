Formulários

Formulários são elementos HTML que permitem a coleta de informações do usuário. Um formulário HTML é definido pela tag <form>, que pode incluir atributos como action e method. O atributo action especifica para onde os dados do formulário serão enviados, enquanto o atributo method define o método HTTP usado para enviar os dados (geralmente GET ou POST).

Dentro do formulário, usamos a tag <input> para criar campos de entrada. Por exemplo, um campo de entrada do tipo texto é definido com <input type="text" name="nome">, onde type="text" especifica que o campo é de texto e name="nome" define o nome do campo, que será usado para identificar os dados enviados.

Para enviar o formulário, usamos um botão de envio, definido com <input type="submit" name="enviar">. O atributo type="submit" cria um botão que, quando clicado, envia os dados do formulário, e o atributo name="enviar" define o nome do botão de envio. 

        
          <form action="https://www.youtube.com/results">
            <input method="GET" type="text" name="search_query">
            <input type="submit" value="pesquisar no Youtube">
          </form>

Nesse exemplo, temos um formulário simples que coleta um texto e faz uma requisição para o youtube usando o método GET. quando o botão de envio é clicado uma pesquisa é feita no youtube com o texto inserido no campo de texto.

Tipos de Input

Existem vários tipos de inputs que podem ser usados em formulários HTML, cada um com uma finalidade específica:

Senha

O input do tipo password é usado para campos onde o usuário deve digitar uma senha. O texto digitado é mascarado para garantir a privacidade. É importante não usar o método GET ao enviar formulários que contenham senhas, pois os dados enviados pelo método GET são visíveis na URL, o que pode comprometer a segurança.

Checkbox

O input do tipo checkbox permite que o usuário selecione uma ou mais opções de um conjunto. Cada checkbox pode ter um atributo value que define o valor enviado quando o checkbox está marcado.

Radio

O input do tipo radio permite que o usuário selecione apenas uma opção de um conjunto. Todos os inputs do tipo radio que pertencem ao mesmo grupo devem ter o mesmo atributo name. Cada opção deve ter um atributo value que define o valor enviado quando a opção está selecionada. O atributo checked pode ser usado para definir qual opção está selecionada por padrão.

Reset

O input do tipo reset cria um botão que, quando clicado, redefine todos os campos do formulário para seus valores iniciais. 

        
          <form>
            Senha <input type="password" name="senha">
            <input type="checkbox" name="opcao1" value="opcao1"> Opção 1 <br>
            <input type="checkbox" name="opcao2" value="opcao2"> Opção 2 <br>
            <input type="radio" name="opcao" value="opcao1"> Masculino
            <input type="radio" name="opcao" value="opcao2"> Feminino
            <input type="radio" name="opcao" value="opcao3" checked> Outro <br>
            <input type="reset" value="Limpar">
          </form>

Select

A tag <select> é usada para criar uma lista suspensa em um formulário HTML. O atributo name define o nome do campo que será enviado quando o formulário for submetido. O atributo size especifica o número de opções visíveis na lista suspensa. O atributo multiple permite que o usuário selecione mais de uma opção na lista.

Dentro da tag <select>, usamos a tag <option> para definir cada opção da lista. O atributo value de <option> define o valor que será enviado quando a opção for selecionada. O atributo selected pode ser usado para definir qual opção está selecionada por padrão. 

        
          <select name="opcoes">
            <option value="opcao1">Opção 1</option>
            <option value="opcao2">Opção 2</option>
            <option value="opcao3">Opção 3</option>
            <option value="opcao4" selected>Opção 4</option>
          </select>
          <select name="opcoes" size="4" multiple>
            <option value="opcao1">Opção 1</option>
            <option value="opcao2" selected>Opção 2</option>
            <option value="opcao3">Opção 3</option>
            <option value="opcao4">Opção 4</option>
          </select>

Textarea

A tag <textarea> é usada para criar um campo de texto de várias linhas em um formulário HTML. O atributo name define o nome do campo que será enviado quando o formulário for submetido. O atributo rows especifica o número de linhas visíveis no campo, e o atributo cols especifica o número de colunas visíveis. 

        
          <textarea name="comentario" rows="4" cols="50">
            Escreva seu comentário aqui...
          </textarea>

Novos Inputs com HTML5

O HTML5 introduziu vários novos tipos de inputs que facilitam a coleta de dados específicos e melhoram a experiência do usuário. Aqui estão alguns dos novos tipos de inputs:

Color

O input do tipo color permite que o usuário selecione uma cor a partir de um seletor de cores. O valor enviado é um código hexadecimal representando a cor escolhida.

Date

O input do tipo date permite que o usuário selecione uma data a partir de um calendário. Os atributos min e max podem ser usados para definir o intervalo de datas permitidas.

Datetime-local

O input do tipo datetime-local permite que o usuário selecione uma data e hora local. Assim como o tipo date, os atributos min e max podem ser usados para definir o intervalo permitido.

File

O input do tipo file permite que o usuário selecione arquivos para upload. É recomendado usar o método POST para enviar arquivos, pois o método GET não é seguro para transferir dados binários e pode resultar em URLs muito longas.

Number

O input do tipo number permite que o usuário insira um valor numérico. Os atributos min e max podem ser usados para definir o intervalo de valores permitidos.

Range

O input do tipo range permite que o usuário selecione um valor dentro de um intervalo usando um controle deslizante. Os atributos min, max e value podem ser usados para definir o intervalo e o valor padrão.

Tel

O input do tipo tel permite que o usuário insira um número de telefone. Este tipo de input é vantajoso em dispositivos móveis, pois exibe um teclado numérico, facilitando a digitação do número.

Email

O input do tipo email permite que o usuário insira um endereço de email. Este tipo de input realiza verificações básicas para garantir que o valor inserido esteja no formato de um endereço de email válido. 

        
          <form>
            Cor: <input type="color" name="cor"><br>
            Data: <input type="date" name="data"><br>
            Data e Hora: <input type="datetime-local" name="datahora"><br>
            Arquivo: <input type="file" name="arquivo"><br>
            Número: <input type="number" name="numero" min="1" max="10"><br>
            Intervalo: <input type="range" name="intervalo" min="0" max="100"><br>
            Telefone: <input type="tel" name="telefone"><br>
            Email: <input type="email" name="email"><br>
            <input type="reset" value="Limpar">
          </form>

Tags Semanticas do HTML5

O HTML5 introduziu várias novas tags semânticas que ajudam a definir a estrutura e o significado do conteúdo de uma página web. Aqui estão algumas das principais tags semânticas:

header

A tag <header> é usada para definir o cabeçalho de uma seção ou página. O cabeçalho geralmente contém elementos como logotipos, títulos, menus de navegação e outros elementos introdutórios.

nav

A tag <nav> é usada para definir uma seção de navegação. Esta tag é usada para agrupar links de navegação importantes, como menus de site ou links de navegação interna.

main

A tag <main> é usada para definir o conteúdo principal de um documento. O conteúdo dentro desta tag deve ser único e relevante para o documento, excluindo cabeçalhos, rodapés e barras laterais.

section

A tag <section> é usada para definir seções temáticas dentro de um documento. Cada seção deve ser identificada por um título e pode conter subseções.

article

A tag <article> é usada para definir um conteúdo independente e auto-contido, como artigos de notícias, postagens de blog, comentários ou qualquer outro conteúdo que possa ser distribuído de forma independente.

footer

A tag <footer> é usada para definir o rodapé de uma seção ou página. O rodapé geralmente contém informações de rodapé, como direitos autorais, links de contato, mapas do site e outros elementos relacionados.
