Keyframes

A propriedade @keyframes é usada para criar animações em CSS. Ela permite que você defina um conjunto de estilos que serão aplicados a um elemento em diferentes pontos do tempo durante a animação.

A sintaxe básica para definir uma animação com @keyframes é a seguinte: 

        
          @keyframes nome-da-animacao {
            0% {
              /* estilos iniciais */
            }
            100% {
              /* estilos finais */
            }
          }

Você pode usar porcentagens para definir os pontos intermediários da animação. Por exemplo: 

        
          @keyframes exemplo {
            0% {
              opacity: 0;
            }
            50% {
              opacity: 0.5;
            }
            100% {
              opacity: 1;
            }
          }

Para aplicar a animação a um elemento, você usa a propriedade animation: 

        
          .meu-elemento {
            animation: exemplo 2s infinite;
          }

No exemplo acima, a animação exemplo será aplicada ao elemento com a classe meu-elemento, durando 2 segundos e repetindo infinitamente.

Propriedades de Animação

Existem várias propriedades que você pode usar para controlar a animação:

  • animation-name: Nome da animação definida com @keyframes.
  • animation-duration: Duração da animação (ex: 2s para 2 segundos).
  • animation-timing-function: Função de tempo da animação:
    • linear: A animação tem uma velocidade constante.
    • ease: A animação começa devagar, acelera no meio e desacelera no final.
    • ease-in: A animação começa devagar e acelera no final.
    • ease-out: A animação começa rápida e desacelera no final.
    • ease-in-out: A animação começa e termina devagar, mas é rápida no meio.
  • animation-delay: Atraso antes do início da animação (ex: 2s para 2 segundos).
  • animation-iteration-count: Número de vezes que a animação será repetida:
    • infinite: A animação será repetida indefinidamente.
    • n: Um número específico de repetições (ex: 3 para repetir 3 vezes).
  • animation-direction: Direção da animação:
    • normal: A animação é executada na direção normal.
    • reverse: A animação é executada na direção inversa.
    • alternate: A animação alterna entre a direção normal e inversa a cada ciclo.
    • alternate-reverse: A animação alterna entre a direção inversa e normal a cada ciclo.
  • animation-fill-mode: Define o estilo aplicado ao elemento quando a animação não está em execução:
    • none: Nenhum estilo é aplicado fora do tempo de execução da animação.
    • forwards: O estilo final da animação é aplicado após a conclusão.
    • backwards: O estilo inicial da animação é aplicado antes do início.
    • both: Ambos os estilos inicial e final são aplicados.
  • animation-play-state: Estado da animação:
    • running: A animação está em execução.
    • paused: A animação está pausada.

Exemplo Completo

Aqui está um exemplo completo de uma animação usando @keyframes e as propriedades de animação: 

          
            @keyframes mover {
              0% {
                left: 0;
              }
              50% {
                transform: translateX(-100%);
                left: 100%;
              }
              100% {
                left: 0;
              }
            }
            
            .animacao {
              position: relative;
              width: 50px;
              height: 50px;
              background-color: red;
              animation: mover 3s ease-in-out infinite;
            }

Nesse exemplo usamos a propriedade animation mesclando as propriedades animation-name, animation-duration, animation-timing-function e animation-iteration-count.

Transitions

A propriedade transition é usada para criar transições suaves entre os estados de um elemento. Ela permite que você defina a duração, o tempo e outros aspectos da transição.

A sintaxe básica para definir uma transição é a seguinte: 

        
          .meu-elemento {
            transition: propriedade duração função-tempo atraso;
          }

Por exemplo, para criar uma transição suave na propriedade background-color de um elemento, você pode usar: 

        
          .meu-elemento {
            transition: background-color 2s ease-in-out;
          }

No exemplo acima, a transição da propriedade background-color durará 2 segundos e usará a função de tempo ease-in-out.

Propriedades de Transição

Existem várias propriedades que você pode usar para controlar a transição:

  • transition-property: A propriedade que será animada (ex: background-color).
  • transition-duration: Duração da transição (ex: 2s para 2 segundos).
  • transition-timing-function: Função de tempo da transição:
    • linear: A transição tem uma velocidade constante.
    • ease: A transição começa devagar, acelera no meio e desacelera no final.
    • ease-in: A transição começa devagar e acelera no final.
    • ease-out: A transição começa rápida e desacelera no final.
    • ease-in-out: A transição começa e termina devagar, mas é rápida no meio.
  • transition-delay: Atraso antes do início da transição (ex: 2s para 2 segundos).

Exemplo Completo

Aqui está um exemplo completo de uma transição usando a propriedade transition: 

          
            .caixa {
              width: 100px;
              height: 100px;
              background-color: blue;
              transition: background-color 2s ease-in-out, transform 2s ease-in-out;
            }

            .caixa:hover {
              background-color: green;
              transform: rotate(45deg);
            }

Nesse exemplo, a cor de fundo da caixa mudará de azul para verde e a caixa girará 45 graus quando o mouse passar sobre ela, com uma transição suave de 2 segundos.
