Estrutura Básica do HTML5

Veja abaixo um exemplo da estrutura fundamental de um documento HTML5:

        
          <html lang="pt-br">
            <head>
              <meta charset="UTF-8" />
              <meta name="viewport" content="width=device-width, initial-scale=1.0" />
              <title>Exemplo de código HTML5</title>
            </head>
            <body>
              <h1>Titulo</h1>
              <p>Isso é um parágrafo.</p>
              <!-- comentario -->
            </body>
          </html>

Explicação dos Elementos

  • <html lang="pt-br">: Define o início do documento HTML e especifica o idioma do conteúdo (lang="pt-br" indica que a página está em português do Brasil).
  • <head>: Contém metadados e informações essenciais para a página, como título, estilos e links.
  • <meta charset="UTF-8" />: Define a codificação de caracteres como UTF-8, garantindo suporte a acentos e caracteres especiais.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0" /> : Ajusta o comportamento da página em diferentes dispositivos.
    • width=device-width: Define a largura da página igual à largura do dispositivo.
    • initial-scale=1.0: Garante que a página seja carregada com zoom padrão de 100%.
  • <title>Exemplo de HTML5</title>: Define o título exibido na aba do navegador.
  • <body>: Contém o conteúdo visível da página, como textos, imagens e links.

Paragráfos, linhas, quebras e comentários

Isso é um paragráfo, é feito usando a tag <p>texto do parágrafo</p>

Um paragrafo pode ser quebrado usando a tag <br> e linhas horizontais são feitas usando a tag <hr> porém é mais indicado que isso seja feito usando CSS

Comentários são Textos no código que não aparecem na página (são feito para desenvolvedores lerem). Em html, comentários são feito assim: <!-- seu comentário -->

Em html podemos fazer algumas formatações:

Negrito: <b>texto</b>, tambem podemos usar a tag <strong>texto</strong>, ambas tem o mesmo resultado visual porem usando strong acrescenta um pouco mais de semântica (dizendo que o texto tem mais força) para o navegador e mecanismos de busca.

Italico: <i>texto</i> e tambem podemos usar a tag <em>texto</em>, ambas tem o mesmo resultado visual porém usando em acrescenta um pouco mais de semântica (dizendo que o texto tem ênfase) para o navegador e mecanismos de busca.

Marcação: <mark>texto</mark>, usado para deixar o texto marcado

nota de rodapé:: <small>texto</small> Usado para deixar o texto um pouquinho menor, pode ser usado em notas de rodapé

Subscrito: H<sub>2</sub>O Usado para deixar o texto subscrito (H2O)

Superscrito: 2<sup>2</sup>=4 Usado para deixar o texto subscrito (22=4)

Links

Isso é um Link: Links são como meios de navegação entre páginas, cada link possuí um endereço pra onde quer ir, e um texto. assim:
<a href="./endereco">texto do link</a>

Esse link abre o google em uma nova aba: pra que isso seja feito devemos colocar o parâmetro target="_blank" assim:
<a href="https://www.google.com" target="_blank">texto do link</a>

Tel: +55 67 9 9200-1234: Há a possibilidade de fazermos um link para um numero de telefone, pra isso precisamos colocar tel: antes do número no endereço do link ficando assim:
<a href="tel:+5567992001234">+55 67 9 9200-1234:</a>

Whatsapp: +55 67 9 9200-1234: O whatapp tem a possibilidade de fazer links que direcionam para uma conversa, pra isso precisamos usar o padrão https://wa.me/numero-do-whatsapp, o código fica assim:
<a https://wa.me/5567992001234" target="_blank">+55 67 9 9200-1234:</a>

Instagram: @matheus.rcg O instagram também tem a possibilidade de fazer links que direcionam direto para o aplicativo (caso o tenha instalado no dispositivo), pra isso usamos um link personalizado que fica assim:
<a href="intagram://user?username=nome-do-usuario">texto do link</a>

Email: meu.email@email.com Pra fazermos links para email, acrescentamos mailto: ao endereço do link, isso fará o usuario ser direcionado ao app padrão de abrir emails, o código fica assim:
<a href="mailto:endereco_do_email@email.com">texto do email</a>

Imagens

Imagens são inseridas através da tag img, nela temos o parâmetro src que será passado o endereço da imagem (seja da internet ou dos próprios arquivos),

outro parâmetro a ser passado é o alt que é um texto alternativo normalmente descrevendo a imagem, esse texto aparecerá no caso do endereço não ser encontrado.

podemos tambem passar width e height para redimencionar a imagem (porém é mais indicado que isso seja feito usando CSS) <img src="./images/html-logo.png" alt="Logo do HTML"width="75px">

Imagens com legendas

Imagens com legendas são feitas usando a tag <figure>conteúdo</figure> Dentro dessa tag colocamos a imagem com a tag img e a legenda é feita com <figcaption>legenda</figcaption>. 

          
            <figure>
              <img src="./images/html-logo.png" alt="Logo do HTML" width="100px">
              <figcaption>
                Texto da legenda
              </figcaption>
            </figure>

Imagens com links

É possivel também inserir uma imagem com link, pra isso precisamos colocar a tag <img> no conteúdo da tag <a>, o codígo ficaria assim: 

            
              <a href="./endereco/do-link">
                <img src="./endereco/da/imagem.extencao" alt="texto alternativo">
              </a>

Listas em HTML

O HTML possui dois tipos principais de listas: ordenadas e não ordenadas.

Tipos de Listas

  • Listas ordenadas: Criadas com a tag <ol>, onde os itens são numerados automaticamente.
  • Listas não ordenadas: Criadas com a tag <ul>, onde os itens são marcados com símbolos.
  • Itens de lista: Em ambas, os itens são definidos com a tag <li>conteúdo</li>.

Personalização de Listas Ordenadas

  • Inversão da ordem: Usando o atributo reversed, os itens são exibidos em ordem decrescente.
    Exemplo: <ol reversed>itens</ol>
  • Definir o início da numeração: O atributo start define o número inicial da contagem.
    Exemplo: <ol start="15">itens</ol>
  • Definir o tipo de marcador: O atributo type permite escolher diferentes estilos de numeração.
    Exemplo: <ol type="a">itens</ol>

Tipos de numeração

  • type="A": Alfabeto maiúsculo (A, B, C...)
  • type="a": Alfabeto minúsculo (a, b, c...)
  • type="I": Números romanos maiúsculos (I, II, III...)
  • type="i": Números romanos minúsculos (i, ii, iii...)
  • Padrão: Numeração decimal (1, 2, 3...)

Hierarquia em Listas

Para criar listas aninhadas (subníveis), basta inserir uma lista dentro de outra.

          
            <ul>
              <li>
                Item principal
                <ul>
                  <li>Item dentro do item</li>
                </ul>
              </li>
            </ul>

Resultado:

  • Item principal
    • Item dentro do item

DIV's e ID's

A tag <div> cria um contêiner genérico que é utilizado para agrupar outros elementos HTML e aplicar estilos ou scripts a eles de forma conjunta. "Div" é uma abreviação de "division" (divisão)

Id's é um parâmetro adicionado a elementos html para que estes elementos possam ser posteriormente referenciados

Aqui temos um exemplo de uma <div> junto com um id que contem uma lista de links que referenciam uma <div> pelo seu id 

          
            <div id="indice">
              <ul>
                <li>
                  <a href="#nome_do_id">texto de referência</a>
                </li>
                <li>
                  <a href="#div_exemplo">Div Exemplo</a>
                </li>
              </ul>
            </div>
            <div id="nome_do_id">
              <h3>um titulo qualquer</h3>
            </div>
            <div id="div_exemplo">
              <h3>Div Exemplo</h3>
            </div>

esse é o resultado: (foi acrescentado texto as divs para melhor compreenção)

  • texto de referência
  • Div Exemplo
