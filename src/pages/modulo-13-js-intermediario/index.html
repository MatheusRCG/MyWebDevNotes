Windown

Windown é o objeto global do navegador, ele é o objeto que contém todos os outros objetos do navegador, como o document, history, location, navigator, screen, alert, confirm, prompt, etc.

Quando você declara uma variável sem o uso da palavra chave var, let ou const, ela é automaticamente atribuída ao objeto global, ou seja, ao objeto window. 

        
          x = 10;
          console.log(window.x); // 10

document

O objeto document é um objeto que representa o documento HTML que está sendo exibido no navegador. Ele é um objeto filho do objeto window.

O objeto document é o principal objeto do DOM (Document Object Model), ele é o responsável por manipular o conteúdo HTML da página. 

        
          console.log(document); // document

Acessando Elementos

getElementById

Para acessar um elemento HTML através do JavaScript, você pode utilizar o método getElementById(), que recebe como parâmetro o id do elemento que você deseja acessar. 

          
            const element = document.getElementById('elementId');

O método getElementById() retorna o elemento HTML que possui o id especificado, ou null caso não exista nenhum elemento com o id especificado.

getElementsByClassName

Você também pode acessar um elemento HTML através de sua classe utilizando o getElementsByClassName(). 

          
            const elements = document.getElementsByClassName('className');

getElementsByTagName

Outra forma de acessar um elemento HTML, é através de sua tag html utilizando o getElementsByTagName(). 

          
            const elements = document.getElementsByTagName('tag');

O método getElementsByTagName() retorna uma coleção de elementos HTML que possuem a tag especificada, ou uma coleção vazia caso não exista nenhum elemento com a tag especificada.

Modificando elementos

Para modificar um elemento HTML, você pode utilizar as propriedades do objeto Element, como innerHTML, innerText, textContent, etc. 

        
          const element = document.getElementById('elementId');
          element.innerHTML = 'Novo conteúdo';

O código acima altera o conteúdo HTML do elemento com o id elementId para 'Novo conteúdo'.

A diferença entre as propriedades innerHTML, innerText e textContent é que a propriedade innerHTML retorna o conteúdo HTML do elemento, a propriedade innerText retorna o conteúdo de texto do elemento, ja a propriedade textContent retorna o conteúdo de texto do elemento incluindo os elementos filhos.

Você também pode modificar o estilo de um elemento HTML utilizando a propriedade style do objeto Element. 

        
          const element = document.getElementById('elementId');
          element.style.color = 'red';

O código acima altera a cor do texto do elemento com o id elementId para vermelho.

eventos

Eventos são ações que ocorrem em um elemento HTML, como um clique, um hover, um scroll, etc. Você pode adicionar eventos a um elemento HTML utilizando o método addEventListener(). 

        
          const element = document.getElementById('elementId');
          element.addEventListener('click', () => {
            console.log('Clicou no elemento');
          });

O código acima adiciona um evento de clique ao elemento com o id elementId, que exibe a mensagem 'Clicou no elemento' no console quando o elemento é clicado.

Outra forma de fazer o codigo acima é passando uma função como segundo parâmetro do método, ao invés de uma arrow function. 

        
          const element = document.getElementById('elementId');
          element.addEventListener('click', printar()) 
          
          function printar() {
            console.log('Clicou no elemento');
          };

Usando o this dentro de um evento, você pode acessar o elemento que disparou o evento. Isso é útil quando você deseja acessar o elemento que disparou o evento dentro da função que trata o evento. 

        
          const element = document.getElementById('elementId');
          element.addEventListener('click', maiusculo) 
          
          function maiusculo() {
            this.innerHTML = this.innerHTML.toUpperCase();
          };

O código acima adiciona um evento de clique ao elemento com o id elementId, que transforma o texto do elemento em maiúsculo quando o elemento é clicado. Usando o this, você pode acessar o elemento que disparou o evento. Nesse caso, o this se refere ao elemento com o id elementId.

Podemos também adicionar eventos a um elemento HTML é utilizando os atributos de eventos HTML, como onclick, onmouseover, onscroll, etc.. Porém, essa forma não é recomendada pois mistura o HTML com o JavaScript.

Elementos Pai e Filhos

Para acessar o elemento pai de um elemento HTML, você pode utilizar a propriedade parentElement do objeto Element. 

        
          const element = document.getElementById('elementId');
          const parent = element.parentElement;

Para acessar os elementos filhos de um elemento HTML, você pode utilizar a propriedade children do objeto Element. 

        
          const element = document.getElementById('elementId');
          const children = element.children;

Biblioteca Math

A biblioteca Math é uma biblioteca nativa do JavaScript que contém métodos matemáticos para realizar operações matemáticas.

Alguns métodos da biblioteca Math são:

  • Math.abs(): retorna o valor absoluto de um número.
  • Math.ceil(): arredonda um número para cima.
  • Math.floor(): arredonda um número para baixo.
  • Math.round(): arredonda um número para o inteiro mais próximo.
  • Math.max(): retorna o maior número de uma lista de números.
  • Math.min(): retorna o menor número de uma lista de números.
  • Math.pow(): retorna a base elevada ao expoente.
  • Math.sqrt(): retorna a raiz quadrada de um número.
  • Math.random(): retorna um numero aleatório entre 0 e 1

Para utilizar os métodos da biblioteca Math, você deve chamar o método Math seguido de um ponto e o nome do método. 

        
          const numero = -10;
          console.log(Math.abs(numero)); // 10

          let aleatorio = Math.random()*10; // número aleatório entre 0 e 10
          console.log(Math.round(aleatorio)); // usa math.floor para arredondar o numero

          let elevado = Math.pow(2, 3); // 2 elevado a 3

          let numeros = [10, 20, 30];
          let maximo = Math.max(...numeros); // 30

em Math.max() o spread operator (...) é utilizado para passar os elementos do array como argumentos para o método Math.max() ja que esse método não aceita um array como argumento.

JSON

JSON (JavaScript Object Notation) é um formato de dados que é utilizado para representar objetos JavaScript em formato de texto. É muito utilizado para enviar e receber dados de servidores, pois é um formato de dados leve e fácil de ser lido e interpretado. Além disso, JSON é um formato de dados independente de linguagem, ou seja, pode ser utilizado em qualquer linguagem de programação.

JSON é composto por pares de chave e valor, onde a chave é uma string e o valor pode ser um número, uma string, um booleano, um array, um objeto ou null. 

        
          const pessoa = {
            nome: 'Matheus',
            idade: 25,
            cidade: 'São Paulo'
          };

          const json = JSON.stringify(pessoa);
          console.log(json); // {"nome":"Matheus","idade":25,"cidade":"São Paulo"}

          const objeto = JSON.parse(json);
          console.log(objeto); // {nome: 'Matheus', idade: 25, cidade: 'São Paulo'}

O método JSON.stringify() converte um objeto JavaScript em uma string JSON. O método JSON.parse() converte uma string JSON em um objeto JavaScript.

Persistência de dados

O Local Storage é uma forma de armazenar dados no navegador do usuário. Os dados armazenados no Local Storage ficam disponíveis mesmo após o usuário fechar o navegador ou desligar o computador.

Armazenando

Para armazenar dados no Local Storage, você pode utilizar o método setItem(), que recebe como parâmetro o nome da chave e o valor que você deseja armazenar. 

          
            localStorage.setItem('nome', 'Matheus');

Recuperando

Para recuperar dados do Local Storage, você pode utilizar o método getItem(), que recebe como parâmetro o nome da chave que você deseja recuperar. 

          
            const nome = localStorage.getItem('nome');
            console.log(nome); // Matheus

Removendo

Para remover um item do Local Storage, você pode utilizar o método removeItem(), que recebe como parâmetro o nome da chave que você deseja remover. 

          
            localStorage.removeItem('nome');

Para remover todos os itens do Local Storage, você pode utilizar o método clear(). 

          
            localStorage.clear();

Armazenando Objetos

O Local Storage armazena os dados em pares chave-valor, onde a chave e o valor são strings. Para armazenar objetos no Local Storage, você deve convertê-los para JSON utilizando o método JSON.stringify(). 

          
            const pessoa = {
              nome: 'Matheus',
              idade: 25,
              cidade: 'São Paulo'
            };
  
            localStorage.setItem('pessoa', JSON.stringify(pessoa));

Para recuperar um objeto do Local Storage, você deve converter a string JSON para um objeto JavaScript utilizando o método JSON.parse(). 

          
            const pessoa = JSON.parse(localStorage.getItem('pessoa'));
            console.log(pessoa); // {nome: 'Matheus', idade: 25, cidade: 'São Paulo'}

Timer e Intervalo

O JavaScript possui dois métodos para execut ar código de forma assíncrona: setTimeout() e setInterval().

O método setTimeout() executa uma função após um determinado tempo, que é especificado em milissegundos. 

        
          setTimeout(() => {
            console.log('Executou após 2 segundos');
          }, 2000);

O método setInterval() executa uma função repetidamente em um intervalo de tempo especificado. 

        
          setInterval(() => {
            console.log('Executou a cada 2 segundos');
          }, 2000);

Para parar a execução de um intervalo, você pode utilizar o método clearInterval(), que recebe como parâmetro o ID do intervalo que você deseja parar. 

        
          const intervalo = setInterval(() => {
            console.log('Executou a cada 2 segundos');
          }, 2000);

          clearInterval(intervalo);

            const numero = document.getElementById('numero');
            const iniciar = document.getElementById('iniciar');
            const parar = document.getElementById('parar');
            const zerar = document.getElementById('zerar');

            let segundos = 0;
            let intervalo = null;

            iniciar.addEventListener('click', () => {
              if (intervalo) return;
              intervalo = setInterval(() => {
                segundos++;
                numero.textContent = `${segundos.toString()} segundos`;
              }, 1000);
            });

            parar.addEventListener('click', () => {
              clearInterval(intervalo);
              intervalo = null;
            });

            zerar.addEventListener('click', () => {
              clearInterval(intervalo);
              intervalo = null;
              segundos = 0;
              numero.textContent = `${segundos.toString()} segundos`;
            });
