Atributos

SetAttribute

A função setAttribute é usada para adicionar ou modificar atributos de um elemento HTML. Ela recebe dois parâmetros: o nome do atributo e o valor que você deseja atribuir a ele. 

          
            const elemento = document.getElementById('meuElemento');
            elemento.setAttribute('class', 'minhaClasse');

No exemplo acima, estamos selecionando um elemento com o id meuElemento e adicionando a ele uma classe chamada minhaClasse.

Você também pode usar setAttribute para modificar atributos existentes. Por exemplo, para alterar o valor de um atributo src de uma imagem: 

          
            const imagem = document.getElementById('minhaImagem');
            imagem.setAttribute('src', 'novoCaminho/paraImagem.png');

No exemplo acima, estamos alterando o atributo src de uma imagem para um novo caminho.

A função setAttribute é muito útil quando você precisa manipular dinamicamente os atributos dos elementos HTML usando JavaScript.

getAttribute

A função getAttribute é usada para obter o valor de um atributo de um elemento HTML. Ela recebe um parâmetro: o nome do atributo que você deseja obter. 

          
            const elemento = document.getElementById('meuElemento');
            const classe = elemento.getAttribute('class');
            console.log(classe);

No exemplo acima, estamos selecionando um elemento com o id meuElemento e obtendo o valor de seu atributo class. Em seguida, estamos imprimindo esse valor no console.

A função getAttribute é útil quando você precisa acessar os valores dos atributos dos elementos HTML usando JavaScript.

removeAttribute

A função removeAttribute é usada para remover um atributo de um elemento HTML. Ela recebe um parâmetro: o nome do atributo que você deseja remover. 

          
            const elemento = document.getElementById('meuElemento');
            elemento.removeAttribute('class');

No exemplo acima, estamos selecionando um elemento com o id meuElemento e removendo seu atributo class.

A função removeAttribute é útil quando você precisa remover dinamicamente os atributos dos elementos HTML usando JavaScript.

dataset

A propriedade dataset é usada para acessar os atributos de dados de um elemento HTML. Ela retorna um objeto que contém todos os atributos de dados do elemento. 

          
            const elemento = document.getElementById('meuElemento');
            const valor = elemento.dataset.meuAtributo;
            console.log(valor);

No exemplo acima, estamos selecionando um elemento com o id meuElemento e acessando o valor de seu atributo de dados meuAtributo. Em seguida, estamos imprimindo esse valor no console.

A propriedade dataset é útil quando você precisa acessar os atributos de dados dos elementos HTML usando JavaScript.

Vídeo

Adicionando um vídeo

Para adicionar um vídeo em uma página HTML, você pode usar a tag <video>. Esta tag permite que você incorpore vídeos diretamente em suas páginas web. 

          
            <video src="caminho/para/video.mp4" controls>
              Seu navegador não suporta a tag de vídeo.
            </video>

No exemplo acima, estamos adicionando um vídeo com o caminho especificado e ativando os controles padrão do navegador.

Atributos de Vídeo

A tag <video> suporta vários atributos que permitem controlar o comportamento e a aparência do vídeo. Aqui estão alguns dos principais atributos:

  • src: Especifica o caminho para o arquivo de vídeo.
  • controls: Adiciona controles de reprodução ao vídeo, como play, pause e volume.
  • autoplay: Faz com que o vídeo comece a ser reproduzido automaticamente quando a página é carregada.
  • loop: Faz com que o vídeo seja reproduzido em loop contínuo.
  • preload: Especifica se o vídeo deve ser carregado automaticamente quando a página é carregada. Pode ter os seguintes valores:
    • auto: O vídeo será carregado automaticamente.
    • metadata: Apenas os metadados do vídeo serão carregados.
    • none: O vídeo não será carregado automaticamente.
  • poster: Especifica uma imagem a ser mostrada enquanto o vídeo não está sendo reproduzido.

Exemplo de uso dos atributos:

          
            <video src="caminho/para/video.mp4" controls loop preload="auto" width="100%">
              Seu navegador não suporta a tag de vídeo.
            </video>

Iframe

A tag <iframe> é usada para incorporar conteúdo de outras fontes em uma página HTML. Isso pode incluir outros documentos HTML, vídeos, mapas e muito mais. A tag <iframe> cria uma janela embutida que exibe o conteúdo especificado pelo atributo src.

Aqui está um exemplo básico de uso da tag <iframe>: 

        
      <iframe src="https://www.example.com" width="600" height="400" frameborder="0"></iframe>

video com iframe

Outra maneira de adicionar um vídeo em uma página HTML é usando um <iframe>. Esta tag permite incorporar conteúdo de outros sites, como vídeos do YouTube. 

          
            <iframe src="https://www.youtube.com/embed/VIDEO_ID" width="100%" height="315" frameborder="0" allowfullscreen></iframe>

Áudio

A tag <audio> é usada para incorporar arquivos de áudio em uma página HTML. Isso permite que você adicione música, efeitos sonoros e outros tipos de áudio ao seu site.

a tag source é usada para especificar a origem do arquivo de áudio e o tipo de arquivo. 

        
          <audio controls>
            <source src="./audio/MEDLEY-LEÃO-DE-JUDÁ-MORADA.mp3" type="audio/mpeg">
          </audio>

Canvas

A tag <canvas> é usada para desenhar gráficos, animações e outros elementos visuais em uma página HTML. Você pode usar JavaScript para manipular o conteúdo do <canvas> e criar gráficos interativos e animações.

index.html

        
          <canvas id="meuCanvas" width="200" height="100" style="background-color: white;></canvas>

script.js

        
          const canvas = document.getElementById('canvas-example-squares');
          const ctx = canvas.getContext('2d');
          ctx.moveTo(0, 0);
          ctx.lineTo(100, 50);
          ctx.lineTo(200, 0);
          ctx.strokeStyle= '#0000FF';
          ctx.lineWidth = 3;
          ctx.stroke();
  • getContext('2d'): Retorna um objeto de contexto que permite desenhar no <canvas>.
  • moveTo(x, y): Move o ponto de desenho para as coordenadas especificadas.
  • lineTo(x, y): Desenha uma linha do ponto atual para as coordenadas especificadas.
  • stroke(): Desenha a linha especificada.
  • strokeStyle: Especifica a cor da linha.
  • lineWidth: Especifica a largura da linha em pixels.

Resultado

Desenhando um retângulo

Para desenhar um retângulo em um <canvas>, você pode usar os métodos fillRect(x, y, width, height) e strokeRect(x, y, width, height). 

          
            const canvas = document.getElementById('meuCanvas');
            const ctx = canvas.getContext('2d');
            ctx.fillStyle = '#00FFFF';
            ctx.fillRect(10, 10, 100, 50);
            ctx.strokeStyle = '#0000FF';
            ctx.lineWidth = 3;
            ctx.strokeRect(10, 10, 100, 50);
  • fillStyle: Especifica a cor de preenchimento do retângulo.
  • fillRect(x, y, width, height): Desenha um retângulo preenchido nas coordenadas especificadas.
  • strokeRect(x, y, width, height): Desenha um retângulo sem preenchimento nas coordenadas especificadas.

Inicio e fim do caminho

Para desenhar formas complexas em um <canvas>, você pode usar os métodos beginPath() e closePath(). 

          
            const canvas = document.getElementById('meuCanvas');
            const ctx = canvas.getContext('2d');
            ctx.beginPath();
            ctx.beginPath();
            ctx.moveTo(50, 50);
            ctx.lineTo(100, 50);
            ctx.lineTo(100, 100);
            ctx.closePath();
            ctx.strokeStyle = "#FF0000";
            ctx.lineWidth = 3;
            ctx.stroke();

            ctx.beginPath();
            ctx.moveTo(75, 25);
            ctx.lineTo(150, 75);
            ctx.strokeStyle = "#FF00FF";
            ctx.lineWidth = 3;
            ctx.stroke();
  • beginPath(): Inicia um novo caminho.
  • closePath(): Fecha o caminho atual.

Círculo

Para desenhar um círculo em um <canvas>, você pode usar o método arc(x, y, raio, startAngle, endAngle, anticlockwise). 

          
            const canvas = document.getElementById('meuCanvas');
            const ctx = canvas.getContext('2d');

            let x = 100
            let y = 75
            let radius = 50
            let startAngle = 0
            let endAngle = 2 * Math.PI

            ctx.beginPath();
            ctx.arc(x, y, radius, startAngle, endAngle);
            ctx.fillStyle = '#FF0000';
            ctx.fill();
  • arc(x, y, raio, startAngle, endAngle, anticlockwise): Desenha um arco de círculo nas coordenadas especificadas.
  • fillStyle: Especifica a cor de preenchimento do círculo.
  • fill(): Preenche o círculo com a cor especificada.

Animações

Você pode criar animações em um <canvas> usando JavaScript. Para criar uma animação, você precisa desenhar o conteúdo do <canvas> repetidamente em intervalos regulares.

capturando o elemento e criando o objeto circle

          
            const canvasAnimation = document.getElementById("canvas-example-animation");
            const ctxAnimation = canvasAnimation.getContext("2d");

            let circle = {
              x: 100,
              y: 75,
              radius: 50,
              start: 0,
              end: 0,
              counterclockwise: false,
            };

Função pra desenhar o círculo

          
            function drawCircle(circle) {
              ctxAnimation.beginPath();
              ctxAnimation.rect(0, 0, 200, 150);
              ctxAnimation.fillStyle = "#FFFFFF";
              ctxAnimation.fill();

              ctxAnimation.beginPath();
              ctxAnimation.fillStyle = "#FF00FF";
              ctxAnimation.arc(
                circle.x,
                circle.y,
                circle.radius,
                circle.start,
                circle.end,
                circle.counterclockwise
              );
              ctxAnimation.fill();
            }

Função pra animar o desenho do círculo

          
            setInterval(function () {
              circle.end += 0.5;
            
              if (circle.end >= 2 * Math.PI) {
                circle.end = 0;
                circle.counterclockwise = !circle.counterclockwise;
              }
            
              drawCircle(circle);
            }, 100);

Usamos o setInterval para chamar a função drawCircle a cada 100 milissegundos. Dentro da função, verificamos se o círculo está sendo desenhado no sentido horário ou anti-horário e ajustamos o ângulo de acordo. Quando o círculo atinge 2 * Math.PI (um círculo completo), mudamos a direção do desenho. Isso cria um efeito de animação de círculo aparecendo e sumindo.
