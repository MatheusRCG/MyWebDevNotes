Hello World! JavaScript

O JavaScript é usado em sites para adicionar interatividade e dinamismo. Ele permite criar funcionalidades como validação de formulários, animações, manipulação de elementos HTML e comunicação assíncrona com servidores, tornando a experiência do usuário mais rica e envolvente.

alert

O código abaixo exibe um alerta com a mensagem "Hello World!" ao carregar a página. 

          
            <html>
              <head>
                <script>
                 alert('Hello World!');
                </script>
              </head>
            </html>

Podemos também criar um arquivo JavaScript separado e incluí-lo no HTML usando a tag <script>. Isso é útil para manter o código organizado e reutilizável.

index.html

          
            <html>
              <head>
                <script src="script.js"></script>
              </head>
            </html>

script.js

          
            alert('Hello World!');

console.log

O método console.log() é usado para exibir mensagens no console do navegador. Ele é útil para depurar e testar o código JavaScript. 

          
            <html>
              <head>
                <script>
                  console.log('Hello World!');
                </script>
              </head>
            </html>

Para abrir o console do navegador, pressione F12 ou clique com o botão direito do mouse na página e selecione "Inspecionar" ou "Inspeção".

algoritmo

Um algoritmo é um conjunto de instruções passo a passo para resolver um problema ou realizar uma tarefa. Ele é a base da programação e é usado para criar programas de computador que automatizam processos e tomam decisões com base em dados.

algoritmo de Receita de Miojo

  1. Ferva 500ml de água em uma panela.
  2. Adicione o miojo na água fervente.
  3. Cozinhe por 3 minutos.
  4. Escorra a água.
  5. Adicione o tempero do miojo e misture bem.
  6. Sirva e aproveite.

algoritmo Atravessar a Rua

  1. Pare na calçada e olhe para os dois lados.
  2. Verifique se não há veículos se aproximando.
  3. Se houver faixa de pedestres, utilize-a.
  4. Comece a atravessar a rua com cuidado.
  5. Continue olhando para os dois lados enquanto atravessa.
  6. Chegue ao outro lado da rua em segurança.

algoritmo matemático

  1. Comece com um número inicial.
  2. Some 5 ao número inicial.
  3. Multiplique o resultado por 2.
  4. Subtraia 4 do resultado.
  5. Divida o resultado por 2.
  6. Exiba o resultado final.

Variaveis em JavaScript

As variáveis são usadas para armazenar valores e dados em JavaScript. Elas são declaradas usando a palavra-chave var, let ou const, seguida pelo nome da variável e, opcionalmente, um valor inicial.

Existem diferentes convenções para nomear variáveis em JavaScript:

  • camelCase: A primeira letra é minúscula e cada nova palavra começa com uma letra maiúscula. Exemplo: minhaVariavel.
  • snake_case: Todas as letras são minúsculas e as palavras são separadas por underscores. Exemplo: minha_variavel.
  • PascalCase: Similar ao camelCase, mas a primeira letra também é maiúscula. Exemplo: MinhaVariavel.
  • kebab-case: Todas as letras são minúsculas e as palavras são separadas por hifens. Exemplo: minha-variavel.

Além disso, é importante seguir algumas regras ao nomear variáveis:

  • Os nomes devem começar com uma letra, underscore (_) ou cifrão ($).
  • Os nomes não podem começar com números.
  • Os nomes podem conter letras, números, underscores e cifrões.
  • Os nomes são case-sensitive, ou seja, minhaVariavel e minhavarivel são variáveis diferentes.

Declarando uma variável e reatribuindo um valor

        
          var nome = "João";
          console.log(nome); // Saída: João

          nome = "Maria";
          console.log(nome); // Saída: Maria

Copiando o valor de uma variável para outra

        
          var nome2 = "Pedro";
          var nome3 = nome2;
          console.log(nome3); // Saída: Pedro

Declarando variáveis sem valor inicial e atribuindo valor posteriormente

        
          var pais;
          console.log(pais); // Saída: undefined

          pais = "Brasil";
          console.log(pais); // Saída: Brasil

Tipo de dados

JavaScript é uma linguagem de programação dinamicamente tipada, o que significa que as variáveis não têm um tipo de dados fixo. O tipo de dados de uma variável é determinado automaticamente com base no valor atribuído a ela.

Existem vários tipos de dados em JavaScript, incluindo:

  • String: Sequência de caracteres, como "Hello World!"
  • Number: Números inteiros ou decimais, como 42 ou 3.14
  • Boolean: Valores verdadeiro ou falso, como true ou false
  • Array: Coleção ordenada de valores, como [1, 2, 3]
  • Object: Coleção de pares chave-valor, como {nome: "João", idade: 30}
  • Undefined: Valor atribuído a variáveis não inicializadas
  • Null: Valor nulo, indicando a ausência de valor
          
            var nome = "João"; // String
            var idade = 30; // Number
            var casado = false; // Boolean
            var numeros = [1, 2, 3]; // Array
            var pessoa = {nome: "João", idade: 30}; // Object
            var vazio = undefined; // Undefined
            var nulo = null; // Null

Operações Aritméticas

JavaScript suporta várias operações aritméticas para realizar cálculos matemáticos. Os operadores aritméticos mais comuns são:

  • Soma (+): Adiciona dois valores.
  • Subtração (-): Subtrai um valor de outro.
  • Multiplicação (*): Multiplica dois valores.
  • Divisão (/): Divide um valor por outro.
  • Módulo (%): Retorna o resto da divisão de um valor por outro.
  • Incremento (++): Adiciona 1 a um valor.
  • Decremento (--): Subtrai 1 de um valor.

Exemplo de operações aritméticas em JavaScript: 

        
          var x = 10;
          var y = 5;

          var soma = x + y;
          var subtracao = x - y;
          var multiplicacao = x * y;
          var divisao = x / y;
          var modulo = x % y;
          var incremento = x++;
          var preIncremento = ++x;
          var decremento = y--;
          var preDecremento = --y;

          console.log(soma); // Saída: 15
          console.log(subtracao); // Saída: 5
          console.log(multiplicacao); // Saída: 50
          console.log(divisao); // Saída: 2
          console.log(modulo); // Saída: 0
          console.log(incremento); // Saída: 10
          console.log(decremento); // Saída: 5

Módulo

O operador módulo (%) retorna o resto da divisão de um número pelo outro. Ele é útil para verificar se um número é divisível por outro ou para obter o último dígito de um número. 

          
            var x = 10;
            var y = 5;

            var modulo = x % y;
            console.log(modulo); // Saída: 0

            var modulo = y % x;
            console.log(modulo); // Saída: 5

Incremento e Decremento

Os operadores de incremento (++) e decremento (--) são usados para adicionar ou subtrair 1 de uma variável, respectivamente. Eles são úteis para atualizar o valor de uma variável de forma rápida e concisa. 

          
            var x = 5;

            var incremento = x++;
            console.log(incremento); // x=6 mas a saída: 5

            var preIncremento = ++x;
            console.log(preIncremento); // x=7 e saída: 7

            var y = 5;

            var decremento = y--;
            console.log(decremento); // x=4 mas a saída: 5

            var preDecremento = --y;
            console.log(preDecremento); // x=3 e saída: 3

A diferença entre colocar os operadores de incremento (++) ou decremento (--) antes ou depois da variável está no momento em que a operação é realizada. Quando o operador é colocado antes da variável (pré-incremento ou pré-decremento), o valor da variável é incrementado ou decrementado antes de ser utilizado na expressão. Por outro lado, quando o operador é colocado depois da variável (pós-incremento ou pós-decremento), o valor da variável é utilizado na expressão antes de ser incrementado ou decrementado.

operador de Atribuições

Em JavaScript, os operadores de atribuição combinados são usados para realizar uma operação aritmética e atribuir o resultado à mesma variável. Esses operadores são úteis para simplificar o código e torná-lo mais conciso.

Operador Descrição Exemplo Equivalente a
+= Adição x += 5; x = x + 5;
-= Subtração x -= 5; x = x - 5;
*= Multiplicação x *= 5; x = x * 5;
/= Divisão x /= 5; x = x / 5;
%= Módulo x %= 5; x = x % 5; 
        
          var x = 10;
          x += 5; // Equivalente: x = x + 5
          console.log(x); // Saída: 15

          var y = 10;
          y -= 5; // Equivalente: y = y - 5
          console.log(y); // Saída: 5

          var z = 10;
          z *= 5; // Equivalente: z = z * 5
          console.log(z); // Saída: 50

          var w = 10;
          w /= 5; // Equivalente:  w = w / 5
          console.log(w); // Saída: 2

          var v = 10;
          v %= 3; // Equivalente: v = v % 3
          console.log(v); // Saída: 1

Operadores de comparação

Os operadores de comparação são usados para comparar dois valores e retornar um valor booleano (true ou false) com base no resultado da comparação.

  • == : Igual a
  • === : Estritamente igual a
  • != : Diferente de
  • !== : Estritamente diferente de
  • > : Maior que
  • < : Menor que
  • >= : Maior ou igual a
  • <= : Menor ou igual a
        
          var a = 5;
          var b = "5";
          var c = 10;

          console.log(a == b);  // true
          console.log(a === b); // false
          console.log(a != c);  // true
          console.log(a !== b); // true
          console.log(a > c);   // false
          console.log(a < c);   // true
          console.log(a >= 5);  // true
          console.log(a <= 4);  // false

O operador === (estritamente igual a) compara dois valores e retorna true se eles são iguais em valor e tipo. Por exemplo, 5 === 5 retorna true, mas 5 === "5" retorna false porque os tipos são diferentes. O operador !== (estritamente diferente de) compara dois valores e retorna true se eles são diferentes em valor ou tipo. Por exemplo, 5 !== "5" retorna true porque os tipos são diferentes, enquanto 5 !== 5 retorna false porque os valores e tipos são iguais.

Operadores Lógicos

Os operadores lógicos são usados para combinar expressões booleanas e retornar um valor booleano com base no resultado da combinação. Os operadores lógicos mais comuns são:

  • && : E lógico
  • || : OU lógico
  • ! : NÃO lógico
        
          var a = true;
          var b = false;

          console.log(a && b); // false
          console.log(a || b); // true
          console.log(!a);     // false
          console.log(!b);     // true

Strings

Uma string é uma sequência de caracteres, como "Hello World!". Em JavaScript, as strings são representadas por valores entre aspas simples (' ') ou aspas duplas (" ").

Existem várias operações que podem ser realizadas em strings, como concatenação, extração de substrings, busca de padrões e substituição de texto.

Concatenação de Strings

A concatenação de strings é a operação de combinar duas ou mais strings em uma única string. Em JavaScript, a concatenação de strings é feita usando o operador de adição (+). 

          
            var nome = "João";
            var sobrenome = "Silva";
            var nomeCompleto = nome + " " + sobrenome;
            console.log(nomeCompleto); // Saída: João Silva

Comprimento de uma String

O comprimento de uma string é o número de caracteres que ela contém. Em JavaScript, o comprimento de uma string pode ser obtido usando a propriedade length. 

          
            var texto = "Hello World!";
            console.log(texto.length); // Saída: 12

Extração de Substrings

Uma substring é uma parte de uma string. Em JavaScript, as substrings podem ser extraídas usando o método substring(pos inicial, pos final) ou a notação de colchetes []. 

          
            var texto = "Hello World!";
            var substring = texto.substring(6, 11);
            console.log(substring); // Saída: World
  
            var primeiraLetra = texto[0];
            console.log(primeiraLetra); // Saída: H

Busca de Padrões

A busca de padrões em uma string é feita usando o método indexOf(). O método indexOf() retorna a posição da primeira ocorrência de um padrão na string, caso contrário retorna -1. É possivel também usar um segundo argumento para indicar a posição inicial da busca. 

          
            var texto = "Hello World!";
            var posicao = texto.indexOf("World");
            console.log(posicao); // Saída: 6
            
            var posicao2 = texto.indexOf("World", 8); // Começa a busca a partir da posição 7
            console.log(posicao2); // Saída: -1 (não encontrado)

Substituição de Texto

A substituição de texto em uma string é feita usando o método replace(). Este método substitui uma parte da string por outra. 

          
            var texto = "Hello World!";
            var novoTexto = texto.replace("World", "JavaScript");
            console.log(novoTexto); // Saída: Hello JavaScript!

Transformando string em numeros

Em JavaScript, é possível converter uma string em um número usando as funções parseInt() e parseFloat(). 

          
            var numero = "42";
            var numeroInteiro = parseInt(numero);
            console.log(numeroInteiro); // Saída: 42

            var numeroDecimal = parseFloat("3.14");
            console.log(numeroDecimal); // Saída: 3.14

Estruturas Condicionais

As estruturas condicionais são usadas para tomar decisões com base em condições específicas. Em JavaScript, as estruturas condicionais mais comuns são:

  • if: Executa um bloco de código se a condição for verdadeira.
  • else: Executa um bloco de código se a condição for falsa.
  • else if: Executa um bloco de código se a condição anterior for falsa e a nova condição for verdadeira.
  • switch: Executa um bloco de código com base em múltiplas condições.

if

A estrutura if é usada para executar um bloco de código se a condição for verdadeira. 

          
            var idade = 18;
  
            if (idade >= 18) {
              console.log("Você é maior de idade.");
            }

else

A estrutura else é usada para executar um bloco de código se a condição for falsa. 

          
            var idade = 16;
  
            if (idade >= 18) {
              console.log("Você é maior de idade.");
            } else {
              console.log("Você é menor de idade.");
            }

else if

A estrutura else if é usada para executar um bloco de código se a condição anterior for falsa e a nova condição for verdadeira. 

          
            var hora = 14;
  
            if (hora < 12) {
              console.log("Bom dia!");
            } else if (hora < 18) {
              console.log("Boa tarde!");
            } else {
              console.log("Boa noite!");
            }

Operador ternário

O operador ternário é uma forma abreviada de escrever uma estrutura condicional if...else. Ele é usado para atribuir um valor a uma variável com base em uma condição.

a estrutura do operador ternário é: condição ? valorSeVerdadeiro : valorSeFalso 

          
            var idade = 18;
            var mensagem = idade >= 18 ? "Maior de idade" : "Menor de idade";
            console.log(mensagem);

switch

A estrutura switch é usada para executar um bloco de código com base em múltiplas condições. 

          
            var dia = 3;
            var nomeDia;
  
            switch (dia) {
              case 1:
                nomeDia = "Domingo";
                break;
              case 2:
                nomeDia = "Segunda-feira";
                break;
              case 3:
                nomeDia = "Terça-feira";
                break;
              case 4:
                nomeDia = "Quarta-feira";
                break;
              case 5:
                nomeDia = "Quinta-feira";
                break;
              case 6:
                nomeDia = "Sexta-feira";
                break;
              case 7:
                nomeDia = "Sábado";
                break;
              default:
                nomeDia = "Dia inválido";
            }
  
            console.log("Hoje é " + nomeDia);

Estrutura de Repetição

As estruturas de repetição são usadas para executar um bloco de código várias vezes. Em JavaScript, as estruturas de repetição mais comuns são:

  • for: Executa um bloco de código um número específico de vezes.
  • while: Executa um bloco de código enquanto a condição for verdadeira.
  • do...while: Executa um bloco de código pelo menos uma vez e depois enquanto a condição for verdadeira.
  • for...in: Itera sobre as propriedades de um objeto.
  • for...of: Itera sobre os elementos de um objeto iterável, como um array.

for

A estrutura for é usada para executar um bloco de código um número específico de vezes. 

          
            for (var i = 0; i < 5; i++) {
              console.log(i);
            }

while

A estrutura while é usada para executar um bloco de código enquanto a condição for verdadeira. 

          
            var i = 0;
  
            while (i < 5) {
              console.log(i);
              i++;
            }

do...while

A estrutura do...while é usada para executar um bloco de código pelo menos uma vez e depois enquanto a condição for verdadeira. 

          
            var i = 0;
  
            do {
              console.log(i);
              i++;
            } while (i < 5);

for...in

A estrutura for...in é usada para iterar sobre as propriedades de um objeto. 

          
            var pessoa = {nome: "João", idade: 30, cidade: "São Paulo"};
  
            for (var propriedade in pessoa) {
              console.log(propriedade + ": " + pessoa[propriedade]);
            }

for...of

A estrutura for...of é usada para iterar sobre os elementos de um objeto iterável, como um array. 

          
            var numeros = [1, 2, 3, 4, 5];
  
            for (var numero of numeros) {
              console.log(numero);
            }

break e continue

Os comandos break e continue são usados para controlar o fluxo de execução de um loop.

O comando break interrompe a execução do loop e sai dele imediatamente. O comando continue interrompe a execução do loop atual e continua para a próxima iteração. 

          
            for (var i = 0; i < 5; i++) {
              if (i === 3) {
                break;
              }
              console.log(i);
            }
          
        
          
            for (var i = 0; i < 5; i++) {
              if (i === 3) {
                continue;
              }
              console.log(i);
            }

Arrays

Um array é uma coleção ordenada de valores, onde cada valor é identificado por um índice. Em JavaScript, os arrays são objetos que podem armazenar múltiplos valores em uma única variável.

Os arrays em JavaScript são dinâmicos, o que significa que eles podem crescer ou encolher dinamicamente à medida que novos elementos são adicionados ou removidos.

Criando um Array

Um array em JavaScript é criado usando colchetes [] e os elementos são separados por vírgulas. 

          
            var numeros = [1, 2, 3, 4, 5];
            var nomes = ["João", "Maria", "Pedro"];

Acessando Elementos de um Array

Os elementos de um array são acessados usando o índice do elemento, que começa em 0. 

          
            var numeros = [1, 2, 3, 4, 5];
            console.log(numeros[0]); // Saída: 1
            console.log(numeros[2]); // Saída: 3

Alterando Elementos de um Array

Os elementos de um array podem ser alterados atribuindo um novo valor ao índice do elemento. 

          
            var numeros = [1, 2, 3, 4, 5];
            numeros[2] = 10;
            console.log(numeros); // Saída: [1, 2, 10, 4, 5]

Adicionando Elementos a um Array

Novos elementos podem ser adicionados a um array usando o método push(). 

          
            var numeros = [1, 2, 3, 4, 5];
            numeros.push(6);
            console.log(numeros); // Saída: [1, 2, 3, 4, 5, 6]

Removendo Elementos de um Array

Elementos de um array podem ser removidos usando o método pop(). 

          
            var numeros = [1, 2, 3, 4, 5];
            numeros.pop();
            console.log(numeros); // Saída: [1, 2, 3, 4]

Iterando sobre um Array

Os elementos de um array podem ser iterados usando um loop for ou for...of. 

          
            var numeros = [1, 2, 3, 4, 5];
  
            for (var i = 0; i < numeros.length; i++) {
              console.log(numeros[i]);
            }
          
        
          
            var numeros = [1, 2, 3, 4, 5];
  
            for (var numero of numeros) {
              console.log(numero);
            }

Funções

Uma função é um bloco de código projetado para executar uma tarefa específica. Em JavaScript, as funções são definidas usando a palavra-chave function, seguida pelo nome da função, parênteses e um bloco de código entre chaves.

Definindo uma Função

          
            function saudacao() {
              console.log("Olá, Mundo!");
            }

Para executar o código dentro de uma função, você deve chamar a função pelo seu nome seguido de parênteses. 

          
            saudacao(); // Saída: Olá, Mundo!

Funções com Parâmetros

As funções podem receber valores de entrada chamados parâmetros. Esses parâmetros são definidos dentro dos parênteses na definição da função. 

          
            function saudacao(nome) {
              console.log("Olá, " + nome + "!");
            }
  
            saudacao("João"); // Saída: Olá, João!

Funções com Retorno

As funções podem retornar um valor usando a palavra-chave return. O valor retornado pode ser armazenado em uma variável ou usado diretamente. 

          
            function soma(a, b) {
              return a + b;
            }
  
            var resultado = soma(5, 3);
            console.log(resultado); // Saída: 8

funções anônimas

As funções anônimas são funções sem nome que podem ser atribuídas a variáveis ou passadas como argumentos para outras funções. 

          
            var saudacao = function(nome) {
              console.log("Olá, " + nome + "!");
            };
  
            saudacao("Maria"); // Saída: Olá, Maria!

Objetos

Um objeto é uma coleção de propriedades, onde cada propriedade é uma chave-valor. Em JavaScript, os objetos são usados para armazenar dados de forma estruturada e acessá-los facilmente.

Criando um Objeto

Um objeto em JavaScript é criado usando chaves {} e as propriedades são definidas como pares chave-valor. 

          
            var pessoa = {
              nome: "João",
              idade: 30,
              cidade: "São Paulo"
            };

Acessando Propriedades de um Objeto

As propriedades de um objeto podem ser acessadas usando a notação de ponto ou a notação de colchetes. 

          
            var nome = pessoa.nome;
            console.log(nome); // Saída: João

            var idade = pessoa["idade"];
            console.log(idade); // Saída: 30

Alterando Propriedades de um Objeto

As propriedades de um objeto podem ser alteradas atribuindo um novo valor à chave da propriedade. 

          
            pessoa.nome = "Maria";
            console.log(pessoa.nome); // Saída: Maria

            pessoa["idade"] = 25;
            console.log(pessoa["idade"]); // Saída: 25

Métodos de um Objeto

Um método é uma função que é uma propriedade de um objeto. Os métodos são definidos como funções dentro do objeto. 

          
            var pessoa = {
              nome: "João",
              idade: 30,
              
              saudacao: function() {
                console.log("Olá, meu nome é " + this.nome);
              }
            };

            pessoa.saudacao(); // Saída: Olá, meu nome é João

A palavra-chave this é uma referência ao objeto atual e é usada para acessar suas propriedades e métodos.

Construtores

Um construtor é uma função usada para criar novos objetos. Em JavaScript, os construtores são definidos usando a palavra-chave function seguida pelo nome do construtor. 

          
            function Pessoa(nome, idade, cidade) {
              this.nome = nome;
              this.idade = idade;
              this.cidade = cidade;
            }

            var pessoa1 = new Pessoa("João", 30, "São Paulo");
            var pessoa2 = new Pessoa("Maria", 25, "Rio de Janeiro");

A palavra-chave new é usada para criar uma nova instância de um objeto usando um construtor.

ForEach

O método forEach() é usado para executar uma função em cada elemento de um array. 

          
            var numeros = [1, 2, 3, 4, 5];

            numeros.forEach(function(numero) {
              console.log(numero);
            });

Datas

Em JavaScript, as datas são representadas pelo objeto Date. O objeto Date é usado para trabalhar com datas e horas em JavaScript.

Para criar uma nova data, você pode usar o construtor Date() sem argumentos, que cria um objeto Date com a data e hora atuais. 

          
            var dataAtual = new Date();
            console.log(dataAtual);

O objeto Date possui vários métodos para obter e definir os valores de data e hora, como getDate(), getMonth(), getFullYear(), getHours(), getMinutes(), getSeconds(), etc. 

          
            var data = new Date();
            var dia = data.getDate();
            var mes = data.getMonth() + 1;
            var ano = data.getFullYear();
            var hora = data.getHours();
            var minutos = data.getMinutes();
            var segundos = data.getSeconds();

            console.log(dia + "/" + (mes) + "/" + ano + " " + hora + ":" + minutos + ":" + segundos);

O método getMonth() retorna o mês de 0 a 11, onde 0 representa janeiro e 11 representa dezembro. Por isso, é necessário adicionar 1 ao valor retornado para obter o mês correto.

Let e Const

Em JavaScript, as variáveis podem ser declaradas usando as palavras-chave var, let e const. A diferença entre elas está no escopo e na mutabilidade da variável.

  • var: Escopo global, pode ser redeclarada e reatribuída.
  • let: Escopo de bloco, não pode ser redeclarada, mas pode ser reatribuída.
  • const: Escopo de bloco, não pode ser redeclarada e nem reatribuída.
        
          var a = 10;
          const b = 30;

          if (true) {
            a = 20;
            let c = 40;
            // b = 50; // Erro: Assignment to constant variable.
          }

          console.log(a); // Saída: 20
          console.log(b); // Saída: 30
          // console.log(c); // Erro: c is not defined

No exemplo acima, a variável a é declarada com var e pode ser acessada dentro e fora do bloco if. A variável b é declarada com const e não pode ser reatribuída. A variável c é declarada com let e só pode ser acessada no escopo onde foi declarada, no caso dentro do bloco if.

O uso de let e const é recomendado em vez de var para evitar problemas de escopo e mutabilidade.
